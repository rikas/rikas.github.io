About me

Hi, my name is Ricardo Otero.

Born in Coimbra, Portugal, I’m a software engineer based in Lisbon, Portugal.

I have a masters degree in Computer Science but I've been into web development way before that. I taught myself Perl then PHP and ended up falling in love with Ruby.

I'm working mainly with Ruby and Ruby on Rails since 2007, but I always try to learn new things, so I've been more into the inevitable world of Javascript recently (specially React).

I've worked in different companies over the years and I always gravitate towards the startup scene. After co-founding and being the CTO of my own startup for 3 years I now work as a freelance web developer working remote first.

I also teach at Le Wagon, a code bootcamp, in Lisbon. It has been a wonderful experience so far. Helping people to completely change their careers teaching things I love is just perfect!

This site is fully static, running on Jekyll and hosted on GitHub Pages. The source code is publicly available, so if you see something that’s incorrect, please submit a pull request.
